# Vue Design System 文档

**Vue Design System** 是一个用于构建 UI 设计系统（Design System）的开源工具，基于 [Vue.js](https://vuejs.org) 开发。它为你的团队提供了一套经过组织的工具、模式（Pattern）和实践，作为应用开发的基石。

该工具构建于 [Vue.js](https://vuejs.org)、[Vue Styleguidist](https://github.com/vue-styleguidist/vue-styleguidist)、[Vue Webpack 模板（Template）](http://vuejs-templates.github.io/webpack/) 和 [Theo](https://github.com/salesforce-ux/theo) 之上，面向至少具备组件化工作流基础知识以及 HTML、SCSS 和 JavaScript 技能的设计师和前端开发者。

**由 [@arielle](https://bsky.app/profile/arie.ls) 及其他贡献者制作。另请参阅 Vue Design System 的[官方网站](https://vueds.com)，以及[阅读我的文章](https://arie.ls/2018/vue-design-system/)，了解我在启动新设计系统项目时所使用的流程和工作方式。**

## 特性

* 一套相互关联的模式与实践，供你和团队直接使用。
* 经过深思熟虑的术语体系、命名约定和层级结构。
* 自动生成概览，追踪设计系统随时间的演进状况。
* 以 YAML 格式定义的全局设计令牌（Design Token），可在任何组件中使用。
* 自动生成活文档（Living Documentation），支持用户实时编辑。
* 可轻松将设计系统导出为 NPM 依赖包，在其他 Vue.js 项目中复用。
* 创建设计令牌、元素（Element）或模式后，即刻在所有组件中可用。
* 预配置的 Prettier 设置，支持保存时和提交前自动格式化代码。
* 内置实时重载（Live Reloading）、自动添加浏览器前缀（Autoprefixing）、SCSS 及辅助函数，并提供开箱即用的 SVG 和 Webfont 合理默认配置。
* 文档与应用逻辑相互分离，因此你可以公开文档，同时保持应用本身的私有性。
